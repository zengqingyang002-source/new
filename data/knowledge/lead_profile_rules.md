# 用户画像研判规则

用户画像研判规则
新加坡国际本硕升学计划用户画像规则
基本属性
​​1.1 年龄​​
初中毕业生：年龄大致在14 - 16岁左右，对应“2 + 2新加坡定向培养本科班”和“2 + 2 + 1本硕连读”项目。
高二在读、应往届高中毕业或中职中技同等学历毕业生：年龄大约在16 - 19岁，针对“0.5/1 + 2新加坡定向培养本科班”和“0.5/1 + 2 + 1本硕连读”项目。
职高、中专、中职、技校学生：年龄满17岁，适合“6 + 6酒店运营大专就业班”和“9 + 6航空运营大专就业班”。
​​1.2 学历​​
初中学历：对应初中毕业生群体，可参与2 + 2及2 + 2 + 1学制项目。
高中学历（含中职、中技等同等学历）：包括高二在读、应往届高中毕业生，以及中职、中技毕业生，可参与0.5/1 + 2及0.5/1 + 2 + 1学制项目。
专科学历：国内专科毕业生，可考虑一年制专升本项目。
本科学历：国内本科毕业生，可关注一年制本升硕项目。
大专学历（职高、中专、中职、技校毕业获得的大专文凭）：通过6 + 6酒店运营大专就业班和9 + 6航空运营大专就业班可获得相应大专文凭。
地域属性
​​国内范围：项目面向全国招生，包括全国范围内的初中毕业生、高二在读及高中毕业生、职高、中专、中职、技校学生等，无明显的地域限制。
社会属性
​​3.1 家庭经济状况​​
项目强调留学费用低、回报率高，整体四年学费总计约30 - 31万人民币，部分项目如0.5/1 + 2国际本科班学费总计约25 - 26万人民币，还有公费0学费留学的情况，说明潜在客户可能有一定的经济基础来承担相应的学费和生活费用，但也考虑到费用因素，适合中等及以上收入家庭，希望以相对合理的成本为孩子提供优质教育机会的家庭。
3.2 教育需求​​
​​3.2.1 升学需求​​：对于初中毕业生、高中毕业生、职高和中专技校毕业生来说，项目提供了升学的新途径，如初中毕业可通过2 + 2或2 + 2 + 1学制升入本科甚至硕士；高中毕业生可通过0.5/1 + 2或0.5/1 + 2 + 1学制实现升学；国内专科和本科毕业生可通过一年制专升本和本升硕项目快速获得更高学历，说明潜在客户有提升学历、进入更好学校深造的需求。
​​3.2.2就业需求​​：项目中如6 + 6酒店运营大专就业班和9 + 6航空运营大专就业班强调解决就业问题，提供带薪实习和就业推荐，就业薪资较高，适合面临国内严峻升学和就业形势，希望通过留学提升就业竞争力的学生群体，潜在客户有通过获得更好的学历和技能来实现就业的强烈需求。
​​3.3 入学能力​​
入学门槛低，但设置了入学测试，包括笔试（英语及心理测评）和面试（英语口语、沟通能力等），说明潜在客户可能学习成绩不是特别突出，但在英语学习和综合素质方面有一定的提升潜力，或者希望通过相对宽松的入学条件进入优质教育项目。
其他属性
​​4.1 留学意愿​​：潜在客户有出国留学的意愿，对新加坡的教育资源和留学环境有一定的兴趣和认可，希望通过留学获得国际化的教育体验和学历认证。
​​4.2 政策关注度​​：对“一带一路”相关政策有一定的关注，可能认识到“一带一路”倡议下留学项目带来的机遇和优势，如项目与“一带一路、乡村振兴”相关联
中德精英人才共建计划
基本属性
​​年龄：18 - 35岁，这个年龄段的人群通常具有一定的学习能力和适应能力，能够较好地接受德国职业教育的培训模式和要求，并且有足够的时间和精力完成职业培训和后续的职业发展。
​​学历：高中及以上学历，表明潜在客户具备一定的知识基础，能够适应德国职业教育中职业学校的专业课程学习。
社会属性
​​2.1 职业需求​​
​​就业导向：项目强调保就业，完成培训考取职业资格证书后可与实习企业签订正式劳动合同，平均薪资较为可观，说明潜在客户有通过接受职业教育获得稳定工作和收入的需求，希望通过德国的职业培训提升自己在就业市场上的竞争力，获得更好的职业发展机会。
​​升学需求：项目提供可升学的途径，完成培训可申请德国大学，免德福、免APS审核，对于那些有进一步深造意愿，希望在德国接受高等教育的人群具有吸引力，潜在客户可能包括那些希望提升学历层次、拓宽知识面和职业发展路径的人。
​​移民需求：项目提到工作两年后可申请德国永居，对于有移民意愿，希望在德国长期生活和发展的人群有一定的吸引力，潜在客户可能包括那些向往德国的生活环境、社会福利，希望获得德国永久居住权的人群。
​​2.2 技能需求​​
​​动手能力强：德国职业教育注重实践培训，培训生需要在企业接受职业技能方面的专业培训，动手能力强是能够顺利完成企业实训的重要条件，潜在客户可能更倾向于通过实践操作来学习和掌握专业技能。
​​逻辑思维强：无论是企业实践还是职业学校的专业课程学习，都需要一定的逻辑思维能力来理解和掌握知识，解决实际问题，潜在客户需要具备这种能力以适应职业培训的要求。
​​2.3 语言能力​​
​​有一定德语学习能力和表达能力：要求有一定德语学习能力和表达能力，并且需要在国内考取德语B1后才能匹配德国企业并进行面试，说明潜在客户需要具备一定的德语基础和学习潜力，以便能够在德国的学习和工作中进行有效的沟通和交流。
​​2.4 家庭背景与经济状况​​
​​家庭支持：虽然项目提供银行全额低息贷款和全程保姆式服务，但留学德国仍然需要一定的经济支持，包括生活费用等。潜在客户可能需要有一定的家庭经济基础或家庭支持，以确保能够顺利完成职业培训。
​​经济理性：项目具有免学费、享补贴等优势，对于一些经济条件有限，但又希望通过留学提升自己的人群具有吸引力，潜在客户可能更注重教育投资的性价比和回报。
其他属性
​​留学意愿：潜在客户有出国留学的意愿，对德国的职业教育体系和教育资源有一定的了解和认可，希望通过德国的职业培训获得国际化的职业技能和职业资格证书。
​​政策关注度：可能关注德国的职业教育政策、移民政策等相关政策，了解德国在职业培训和人才引进方面的优惠政策和机会，以便更好地规划自己的学习和职业发展。
​​学习态度：愿意参加线下封闭式实训，接受严格的管理和培训，说明潜在客户具有较强的学习意愿和学习态度，能够积极主动地投入到学习和培训中，以获得更好的学习效果和职业发展。
