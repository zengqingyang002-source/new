# 数据库设计

数据库名：

```text
edu_dify_platform
```

## 核心表

| 表名 | 说明 |
| --- | --- |
| sys_user | 用户表，保存客户、学生、员工、管理员账号 |
| crm_lead | 意向客户表 |
| lead_evaluation | 客户画像研判结果 |
| employee_daily_report | 员工日报 |
| student_admin_service | 学生请假和行政服务 |
| student_feedback_ticket | 投诉反馈工单 |
| student_psych_profile | 学生心理画像 |
| student_psych_alert | 心理预警 |
| course_project | 课程和项目 |
| event_lecture | 活动讲座 |
| event_registration | 活动报名 |
| student_score | 学生成绩 |
| report_snapshot | 智能报告 |

## 客户研判结果表

```sql
CREATE TABLE lead_evaluation (
    id INTEGER PRIMARY KEY AUTO_INCREMENT,
    lead_id INTEGER,
    source_type VARCHAR(32),
    extracted_info TEXT,
    matched_project VARCHAR(128),
    singapore_score INTEGER DEFAULT 0,
    germany_score INTEGER DEFAULT 0,
    lead_level VARCHAR(8),
    reasons TEXT,
    missing_fields TEXT,
    suggested_questions TEXT,
    sales_advice TEXT,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

## 状态设计

客户状态：

```text
新增意向 / 跟进中 / 已签约 / 已流失
```

请假状态：

```text
待审批 / 已通过 / 已驳回
```

投诉状态：

```text
待处理 / 处理中 / 已解决
```

心理预警状态：

```text
未处理 / 跟进中 / 已解除
```
