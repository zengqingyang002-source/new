# PPT 大纲

## 1. 封面

题目：基于 Dify 的国际教育智能服务平台

副标题：面向客户咨询、画像研判、员工协同与学生服务的一体化 AI 工作台

## 2. 项目背景

- 留学业务咨询频繁，人工客服压力大。
- 客户资料来源复杂，人工判断画像效率低。
- 员工日报、客户跟进、学生服务流程分散。
- 管理层缺少快速汇总和智能分析。

## 3. 项目目标

- 构建一个可运行的国际教育智能服务平台。
- 打通咨询、研判、CRM、学生服务、报告流程。
- 使用 Dify 提升知识问答、智能判断和报告生成能力。

## 4. 用户角色

- 客户
- 学生
- 员工
- 管理员

## 5. 总体架构

```text
Vue3 前端 -> FastAPI 后端 -> MySQL 数据库
                         -> Dify Chatflow / Workflow
```

## 6. 功能模块

- 客服咨询
- 客户画像研判
- 客户管理 CRM
- 企业助手
- 学生服务
- 智能报告

## 7. Dify 应用设计

- 客服咨询 Chatflow
- 客户研判 Workflow
- 企业助手 Chatflow
- 智能报告 Workflow

## 8. 客户研判规则

- 新加坡项目：年龄、学历、升学需求、预算、英语/留学意愿。
- 德国项目：年龄、学历、德语、就业/职业技能、移民/永居诉求。
- 输出推荐项目、分数、等级、原因和建议。

## 9. 数据库设计

展示核心表：

- sys_user
- crm_lead
- lead_evaluation
- student_feedback_ticket
- student_admin_service
- report_snapshot

## 10. 系统演示

按 20 分钟演示脚本展示。

## 11. 项目亮点

- 业务闭环完整
- 支持多类型文件研判
- Dify 与业务系统结合
- 前后端和数据库完整
- 支持 Docker 一键启动

## 12. 总结与展望

后续可扩展：

- 活动报名
- 语音日报
- NL2SQL
- 更复杂权限
- 数据可视化大屏
- 真实教务和 CRM 系统对接
