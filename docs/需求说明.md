# 项目需求说明

## 项目名称

基于 Dify 的国际教育智能服务平台

## 使用场景

本项目用于毕业设计和课堂演示，模拟国际教育服务机构的真实业务场景。

## 角色

| 角色 | 功能 |
| --- | --- |
| 客户 | 项目咨询、政策查询、费用咨询 |
| 学生 | 请假申请、投诉反馈、服务查询 |
| 员工 | 客户研判、CRM 管理、日报、学生服务处理 |
| 管理员 | 仪表盘、智能报告、经营分析 |

## 必做功能

1. 登录和角色权限
2. 首页仪表盘
3. 客服咨询 Agent
4. 客户画像研判
5. 文件上传研判
6. 客户管理 CRM
7. 企业智能助手
8. 员工日报
9. 学生请假
10. 投诉反馈
11. 心理预警
12. 智能报告
13. 系统设置

## 项目亮点

1. 有完整前后端和数据库，不是单独聊天框。
2. Dify 用于知识库问答、客户研判和报告生成。
3. 支持 PDF、Word、Excel、TXT 多类型客户资料研判。
4. 客户画像有分数、等级、理由、缺失信息和销售建议。
5. 演示链路覆盖“咨询-研判-入库-跟进-服务-报告”。
