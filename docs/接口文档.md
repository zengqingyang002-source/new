# 接口文档

后端地址：

```text
http://localhost:8000
```

所有业务接口统一以 `/api` 开头。

## 认证

### 登录

```http
POST /api/auth/login
```

请求：

```json
{
  "username": "admin",
  "password": "123456"
}
```

返回：

```json
{
  "access_token": "token",
  "token_type": "bearer",
  "user": {}
}
```

后续请求 Header：

```text
Authorization: Bearer <access_token>
```

## 仪表盘

```http
GET /api/dashboard
```

返回客户数、投诉数、请假数、心理预警数和项目分布。

## 客服咨询

```http
POST /api/chat/customer
```

请求：

```json
{
  "message": "新加坡 2+2 国际本科班多少钱？"
}
```

## 企业助手

```http
POST /api/chat/employee
```

请求：

```json
{
  "message": "有哪些投诉还没有处理？"
}
```

## 客户研判

### 文本研判

```http
POST /api/leads/evaluate
```

请求：

```json
{
  "text": "张三，19岁，高中毕业，想去新加坡读本科",
  "source_type": "text",
  "save_to_crm": true
}
```

返回：

```json
{
  "evaluation": {
    "matched_project": "新加坡国际本硕升学计划",
    "singapore_score": 100,
    "germany_score": 45,
    "lead_level": "A",
    "reasons": [],
    "missing_fields": [],
    "suggested_questions": [],
    "sales_advice": ""
  }
}
```

### 文件上传研判

```http
POST /api/leads/upload
```

表单字段：

```text
file: PDF / DOCX / XLSX / TXT
```

## 智能报告

```http
GET  /api/reports
POST /api/reports
```

请求：

```json
{
  "report_type": "customer-analysis"
}
```

支持类型：

- customer-analysis
- daily-summary
- complaint-weekly
- psych-weekly
